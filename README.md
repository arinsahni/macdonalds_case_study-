# McDonald's Market Segmentation Analysis 🍔📊

## Overview
This project performs customer segmentation analysis on McDonald's survey data using machine learning techniques such as PCA and K-Means clustering.

The goal is to identify distinct customer groups based on preferences and sentiment to support targeted marketing strategies.

---

## Key Objectives
- Segment customers based on perception and behavior
- Identify high-value and low-engagement groups
- Provide actionable business insights

---

## Methodology
1. Data preprocessing (binary encoding of responses)
2. Feature scaling
3. Dimensionality reduction using PCA
4. Clustering using K-Means
5. Segment profiling and visualization

---

## Key Findings
- Identified **4 distinct customer segments**
- Clear variation in customer sentiment and demographics
- Certain segments show strong brand affinity while others indicate dissatisfaction

---

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
